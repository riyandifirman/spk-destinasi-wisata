# HirwiGo | Sistem Pendukung Keputusan Destinasi Wisata :computer:

Menggunakan metode logika fuzzy tahani dalam pengambilan keputusan <br>
Ditujukan untuk memenuhi tugas besar mata kuliah Logika Informatika
